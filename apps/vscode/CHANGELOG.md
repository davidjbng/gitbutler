# Changelog

All notable changes to the GitButler VSCode extension will be documented in this file.

## [0.1.0] - 2026-01-03

### Added

- Initial release of GitButler VSCode extension
- Virtual branches tree view in Source Control panel
- Display of commits for each branch with author and date information
- File changes visualization for commits and uncommitted changes
- Stack support for stacked branches
- Upstream status showing commits ahead/behind
- Interactive file opening from the tree view
- Real-time updates when files change
- Integration with GitButler CLI using JSON output mode
- Support for unassigned changes display
